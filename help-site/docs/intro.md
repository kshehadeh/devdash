# Welcome to DevDash

DevDash is a macOS desktop application that eliminates the daily ritual of opening a dozen browser tabs to figure out what needs your attention. It pulls your open pull requests, assigned tickets, engineering metrics, and review queue into a single native app — and keeps everything up to date in the background.

## What DevDash does

DevDash connects to **GitHub**, **Jira**, **Linear**, and **Confluence** through your existing credentials, then gives you:

- A **dashboard** with velocity, merge ratio, review turnaround, and workload health metrics
- A **My Day** view for morning standup context — reviews, tickets, reminders, notifications
- A **Reviews** page showing exactly what's waiting on you and what's happening on your own PRs
- A **Team** overview with per-engineer snapshots for leads and managers
- An in-app **notification center** with native desktop alerts for the events that matter
- A **Reminders** system that optionally syncs with macOS Reminders
- A **Command palette** (⌘K) to search everything instantly
- A **menu bar icon** for quick access without switching windows

Data is synced in the background and cached locally so the app is fast, offline-tolerant, and never blocks your work.

## System requirements

| Requirement | Minimum |
|-------------|---------|
| Operating system | macOS 12 Monterey or later |
| Architecture | Apple Silicon (M-series) or Intel |
| Code integration | GitHub account |
| Work tracking | Jira Cloud or Linear *(optional)* |
| Documentation | Confluence Cloud *(optional)* |

## Get started in five minutes

1. **[Download and install](./getting-started/installation)** — grab the `.dmg` from Releases
2. **[Connect your integrations](./getting-started/connections)** — GitHub, Jira, and/or Linear
3. **[Assign data sources](./getting-started/data-sources)** — choose repos, projects, and teams to follow

Once set up, DevDash syncs automatically and you're ready to go.
